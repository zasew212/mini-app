# Магазин вещей - Однофайловое приложение

Это Flask-приложение, которое работает полностью из одного файла без необходимости в отдельной папке с шаблонами.

## Особенности

- Полностью функциональный интернет-магазин
- Админ-панель для управления товарами
- Фильтрация товаров по категориям, размерам и состоянию
- Встроенная система аутентификации администратора
- Все HTML-шаблоны встроены в Python-файл

## Требования

- Python 3.6+
- Flask

## Установка

1. Убедитесь, что у вас установлен Python 3.6 или выше
2. Установите Flask:
   ```bash
   pip install flask
   ```

## Запуск

1. Скачайте файл `app_single_file.py`
2. Запустите приложение:
   ```bash
   python app_single_file.py
   ```
3. Откройте браузер и перейдите по адресу: http://localhost:5000

## Функции администратора

- Просмотр всех товаров
- Добавление новых товаров
- Редактирование существующих товаров
- Удаление товаров

## Структура приложения

Все HTML-шаблоны встроены в Python-файл в виде строковых переменных:

- `BASE_TEMPLATE` - базовый шаблон с CSS и JavaScript
- `HOME_TEMPLATE` - главная страница с товарами
- `PRODUCT_DETAIL_TEMPLATE` - страница деталей товара
- `ADMIN_LOGIN_TEMPLATE` - страница входа администратора
- `ADMIN_PANEL_TEMPLATE` - панель управления администратора
- `ADD_PRODUCT_TEMPLATE` - форма добавления товара
- `EDIT_PRODUCT_TEMPLATE` - форма редактирования товара

## Преимущества однофайлового подхода

- Простота развертывания - нужно только один файл
- Нет зависимости от файловой системы шаблонов
- Легко поделиться приложением
- Все в одном месте для быстрого редактирования

## Безопасность

- Смените пароль администратора в переменных `ADMIN_USERNAME` и `ADMIN_PASSWORD`
- Используйте надежный секретный ключ в `app.secret_key`
- В продакшене используйте `debug=False` в `app.run()`
